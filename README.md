# read_data_ORAS5
Script to read and visualize Data from the data source ORAS5

1.- read_data.m

Reads the data from dataset_oras5 and plots in 3D. The output files can be used for no_seasonal.m. The Oras5 data source is not included due to privacy and data size concerns. 

2.- no_seasonal.m

Produce a time series of salinity variability over the years. Also, an additional plot showing seasonal information is extracted. 

The script is working for the data set in: salt_z20_equator/salt_eq (it needs to be loaded first)
