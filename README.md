# read_data_ORAS5
Script to read and visualize Data from the data source ORAS5

1.- read_data.m

Reads the data from dataset_oras5, and plot in 3D. The output files can be use for no_seasonal.m

2.- no_seasonal.m

Produce a time series with the variability of the salinity respect to years. Also and additional plot where seasonal information is extracted. 

The script is working for the data set in: salt_z20_equator/salt_eq (it needs to be loaded first)
